# Challenge-Literatura

Java-based application developed to manage and explore a digital library. This project focuses on consuming external APIs to fetch book data and processing it to provide a seamless user experience. It was built as a challenge for the Oracle Next Education program. 

Features
-External API Consumption: Integrated with the Gutendex API to retrieve real-time book and author information. 
-Data Persistence: Uses PostgreSQL/MySQL to store searched books and favorite authors. 
-JSON Processing: Efficiently parses complex JSON responses into Java objects. 
-Advanced Statistics: Includes logic to filter books by language and analyze author timelines. 
-Error Handling: Implemented robust exception management for API requests and database operations. 

Tech Stack
-Language: Java 
-Framework: Spring Boot Database: PostgreSQL 
-Tools: Maven, Jackson (for JSON), Git/GitHub. 

How to run it
1. Clone this repository: git clone https://github.com/S4turn322/Challenge-Literatura 
2. Set up your database environment variables.
3. Run the application from your IDE.
4. Use the interactive console menu to search for books by title or author.
